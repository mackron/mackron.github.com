dr_wav

dr_wav is a simple single-file library written in C for loading WAV files. It's open source and public domain.

You can download it here: dr_wav.h

dr_wav is hosted on github. You can submit issues and feature requests from there.

Refer to dr_wav.h for usage and API documentation.

Features

  • Public domain
  • No dependencies except for the C standard library
  • Everything is contained in a single file
  • Tested and supported internal formats include the following:
    • Unsigned 8-bit PCM
    • Signed 12-bit PCM
    • Signed 16-bit PCM
    • Signed 24-bit PCM
    • Signed 32-bit PCM
    • IEEE 32-bit floating point.
    • IEEE 64-bit floating point.
    • A-law and u-law


Limitations and future plans

  • Wave64 (w64) is not currently supported, but will be added in the future. It will work seamlessly with the existing API.
  • Microsoft ADPCM is not supported and will likely not be added in the near future unless somebody really needs it.

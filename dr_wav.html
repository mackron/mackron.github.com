dr_wav

dr_wav is a simple single-file library written in C for loading WAV files. It's open source and public domain.

You can download it here: dr_wav.h

dr_wav is hosted on github. You can submit issues and feature requests from there.

How to use 


    #define DR_WAV_IMPLEMENTATION
    #include "dr_wav.h"

    ...

    drwav* pWav = drwav_open_file("my_song.wav");
    if (pWav == NULL) {
        ... handle error ...
    }

    float* pDecodedInterleavedSamples = malloc(pWav->totalSampleCount * sizeof(float));
    size_t numberOfSamplesActuallyDecoded = drwav_read_f32(pWav, (size_t)pWav->totalSampleCount, pDecodedInterleavedSamples);

    do_something_with_decoded_samples(pWav, numberOfSamplesActuallyDecoded, pDecodedInterleavedSamples);

drwav_open_file() is a high level helper API. You can also use drwav_open() which is a lower level API which takes a pointer to two callbacks - one for reading data and another for seeking. Take a look at the implementation of drwav_open_file() for an example on how to implement these. You can also use drwav_open_memory() which opens the loader from a block of memory.

A few notes

  • Tested and supported internal formats include the following:
    • Unsigned 8-bit PCM
    • Signed 12-bit PCM
    • Signed 16-bit PCM
    • Signed 24-bit PCM
    • Signed 32-bit PCM
    • IEEE 32-bit floating point.
    • IEEE 64-bit floating point.
    • A-law and u-law
  • Use drwav_read() to return samples in their internal format.

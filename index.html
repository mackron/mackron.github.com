Libraries

dr_flac

dr_flac is an open source library for decoding FLAC files. It's written in C in a single file with no dependencies except for the C standard library, and is released into the public domain.

dr_wav

dr_wav is an open source library for decoding WAV files. It's written in C in a single file with no dependencies except for the C standard library, and is released into the public domain.

dr_mp3

dr_mp3 is an open source library for decoding MP3 files. It's written in C in a single file with no dependencies except for the C standard library, and is released into the public domain.

dr_obj

dr_obj is an open source library for loading Wavefront OBJ model files. It's written in C in a single file with no dependencies except for the C standard library. Public domain.

dr_pcx

dr_pcx is an open source library for decoding PCX image files. It's written in C in a single file with no dependencies except for the C standard library, and is released into the public domain.

dr_fs

dr_fs is a simple library which abstracts file IO to allow one to open files from both the native file system and archive/package files such as Zip files using a common API.

vkbind

vkbind is an API loader for Vulkan. It's written in C in a single file with no dependencies except for the C standard library, and is released into the public domain.

dr_webgen

dr_webgen is a simple library for static website generation. It is not intended to replace more complete solutions like Jekyll, Hugo, etc. but is instead focused on simplicity.

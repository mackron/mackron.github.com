dr_pcx

dr_pcx is a simple single-file library written in C for loading PCX files. It's open source and public domain.

You can download it here: dr_pcx.h

dr_pcx is hosted on github. You can submit issues and feature requests from there.

Refer to dr_pcx.h for usage and API documentation.

Features

  • Public domain
  • No dependencies except for the C standard library
  • Everything is contained in a single file
  • Should support every combination of bit planes and bits per pixel configurations, including the CGA palette.


PCX Viewer

This is a simple program for displaying PCX images. It mainly acts as a test for dr_pcx. There's no GUI - just use the Open With feature in Windows Explorer and it should Just Work.

Download: dr_pcx_viewer.exe

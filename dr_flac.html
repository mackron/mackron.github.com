dr_flac

dr_flac is a simple single-file library written in C for decoding FLAC files. It's open source and public domain.

You can download it here: dr_flac.h

dr_flac is hosted on github. You submit issues and feature requests from there.

How to use: 


    #define DR_FLAC_IMPLEMENTATION
    #include "dr_flac.h"

    ...

    drflac* pFlac = drflac_open_file("my_song.flac");
    if (pFlac == NULL) {
        ... handle error ...
    }

    int32_t* pDecodedInterleavedSamples = malloc(pFlac->totalSampleCount * sizeof(int32_t));
    uint64_t numberOfSamplesActuallyDecoded = drflac_read_s32(pFlac, pFlac->totalSampleCount, pDecodedInterleavedSamples);

    do_something_with_decoded_samples(pFlac, numberOfSamplesActuallyDecoded, pDecodedInterleavedSamples);

A few notes:

  • The 32-bit build is about 1.1x to 1.25x slower than the reference implementation, but the 64-bit build is at about parity.
  • dr_flac is good for decoding native FLAC files, but it doesn't currently support streams without a STREAMINFO header or streams that start in the middle of a frame. I'm planning on adding this at some point, but if you need this let me know and I'll prioritize it.
  • Ogg encapsulation is not supported, but again if you need this let me know.

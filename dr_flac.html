dr_flac

dr_flac is a simple single-file library written in C for decoding FLAC files. It's open source and public domain.

You can download it here: dr_flac.h

dr_flac is hosted on github. You can submit issues and feature requests from there.

Refer to dr_flac.h for usage and API documentation.

Features

  • Public domain
  • No dependencies except for the C standard library
  • Everything is contained in a single file
  • Performance roughly comparable to the reference implementation
  • Supports metadata


Limitations and future plans

  • dr_flac does not properly handle corrupt or perverse FLAC streams. This will be addressed in the future.
  • dr_flac does not work with broadcast stream like internet radio and whatnot. Adding support for this is planned for the future.
